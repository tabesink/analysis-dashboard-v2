- Pending-channel-map uploads now show a completion summary with Assign Channels guidance instead of closing silently.
- Removed shallow/dead client modules: legacy channel-map process feedback, channel-map save/upload pass-through wrappers, and duplicate dashboard derived-task status API surface.
