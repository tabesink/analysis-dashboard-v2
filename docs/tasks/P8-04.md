# P8-04: Add `GET /api/v1/sync/version` endpoint

## Summary

Added an authenticated sync endpoint that exposes the backend monotonic
`data_version` counter for polling clients.

## Changes

- Added `server/routers/sync.py` with:
  - `GET /api/v1/sync/version`
  - requires authenticated user (`CurrentUserDep`)
  - returns `{ "data_version": <int> }` from `db.get_data_version()`
- Registered the router in app wiring:
  - `server/routers/__init__.py`
  - `server/main.py`
- Added route-level tests:
  - `tests/server/routers/test_sync_router.py`
  - verifies unauthenticated requests are rejected (`401`)
  - verifies returned `data_version` increases after a write operation

## Verification

```bash
cd server
uv run pytest ../tests/server/routers/test_sync_router.py ../tests/server/routers/test_auth_routes.py
```
