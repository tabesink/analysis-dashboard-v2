# DMD-20-01 — Reusable Edit Metadata panel

Completed: 2026-06-10

## Summary

Extracted the Edit Metadata tab from `/database/edit` into `EditMetadataPanel`, a scope-driven component that accepts `{ programId, version }` and owns draft state, event prefill, save/copy/reset/restore, admin-only status gating, and metadata-save cache invalidation.

The full-page route keeps side-panel program/version selection and passes the selected scope into the panel via callbacks for selection summary and loading/saving activity.

## Key files

| File | Role |
|------|------|
| `client/src/components/edit-metadata/EditMetadataPanel.tsx` | Reusable panel UI + state |
| `client/src/features/edit-metadata/lib/build-program-version-draft.ts` | Pure draft/selection helpers |
| `client/src/features/edit-metadata/lib/metadata-save-state.ts` | Save enablement predicate |
| `client/src/features/edit-metadata/lib/metadata-copy-paste.ts` | Copy/paste key selection and draft merge |
| `client/src/app/database/edit/page.tsx` | Route wrapper; side panel + other tabs unchanged |
