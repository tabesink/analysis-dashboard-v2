# REF37-07 - Remove database import from active product surface

Date: 2026-06-16

## Behavior changed

- Database import is no longer offered in the admin Database settings workflow.
- Admin database import API routes now return `410 Gone` with replacement guidance instead of starting upload/import behavior:
  - `POST /api/v1/export/database/parquet/upload`
  - `POST /api/v1/export/database/parquet/import/{upload_id}`
  - `DELETE /api/v1/export/database/parquet/upload/{upload_id}`
- Database export remains available and unchanged.
- Database create/connect/delete workflow remains available and unchanged.

## Interfaces changed

- `client/src/components/upload/DatabaseSection.tsx`
  - removed import action props and UI affordance.
  - added explicit replacement copy directing admins to export + create/connect workflow.
- `client/src/components/settings/panels/DatabaseSettingsPanel.tsx`
  - removed import action wiring from active settings panel flow.
  - `useDatabaseOperation` now runs with optional arguments in export-only usage.
- `server/routers/export.py`
  - import upload/start/cancel endpoints now enforce removed capability through `410 Gone` responses.

## Tests added or updated

- Added `client/src/components/upload/DatabaseSection.test.tsx`:
  - verifies active admin database operation UI no longer renders import controls.
  - verifies replacement guidance copy is present.
- Updated `tests/server/routers/test_export_router.py`:
  - verifies admin import upload/start/cancel routes return `410` and replacement guidance.
  - preserves export and database-management route contract assertions.

## Follow-on assumptions for next slice

- Database portability now means export-only in this product surface; restore/migration work should use create/connect management workflow.
- Any remaining import-specific internals can be removed in later cleanup slices without reintroducing UI/API behavior.

## Decisions intentionally unchanged

- No changes to export task contracts.
- No changes to create/connect/delete endpoint contracts.
- No changes to folder upload or event-level upload lanes.
