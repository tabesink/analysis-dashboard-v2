# PU-35-05 — Cumulative-by-channel 2D plot spec

## Behavior added

- Added pure 2D spec builder `buildDamage2DPlotSpec` at `client/src/features/inspect-damage-3d/lib/build-damage-2d-plot-spec.ts`.
- The builder now supports `plotType: 'cumulative_by_channel'` and returns a grouped Reference/Target chart spec.
- Spec generation reads comparison aggregate channel rows, filters by selected channel keys, applies absolute/normalized value mode, and applies linear/log damage scaling.
- Spec output includes:
  - ordered channel categories,
  - Reference/Target series values,
  - stable semantic legend colors,
  - subtitle with value/scale context,
  - y-axis scale metadata,
  - warnings and renderable empty states.

## Interfaces changed

- New exported types:
  - `Damage2DChartKind`
  - `Damage2DPlotSpec`
- New exported function:
  - `buildDamage2DPlotSpec(input)`

## Tests added

- `client/src/features/inspect-damage-3d/__tests__/build-damage-2d-plot-spec.test.ts`
  - validates grouped Reference/Target cumulative values,
  - validates selected-channel ordering/filtering,
  - validates absolute vs normalized mode behavior,
  - validates linear vs log scale transform/domain/ticks,
  - validates semantic legend colors,
  - validates renderable empty states for missing channels and missing aggregates.

## Follow-on assumptions for next slice

- `PU-35-06` can consume `Damage2DPlotSpec` for grouped-bar rendering without recomputing aggregates.
- Renderer may rely on `warnings`/`emptyState` fields for shell-state messaging.

## Decisions intentionally unchanged

- No aggregate semantics changed.
- No 3D cell builder behavior changed.
- No SVG/card rendering integration added in this slice.
