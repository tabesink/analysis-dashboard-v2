# PU-35-07 — Wire 2D card grid with focused 3D card

## Behavior added

- Refactored `DamagePlotView` from a single 3D surface with overlay rail into a 2D-first layout: a 2×2 card grid plus one focused 3D card below.
- Added `Damage2DPlotCard` to route cumulative-by-channel specs to `CumulativeByChannelPlotCard` and render placeholder shell cards for not-yet-implemented plot types.
- Clicking a 2D card updates local focused plot type for the retained single Three.js canvas.
- Removed final UI dependency on `DamagePlotOverlayControls` from the Inspect Damage plot area.
- Moved the 3D render-cap warning into the focused 3D card; Normal/Log toggle and color legend remain in the plot area and affect the focused 3D view.

## Interfaces changed

- New exported component: `Damage2DPlotCard`
- `DamagePlotView` layout/state:
  - local `focusedPlotType` replaces overlay-rail plot-type selection
  - builds all four 2D specs via `buildDamage2DPlotSpec`
  - version slice UI removed from plot area
