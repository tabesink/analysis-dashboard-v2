# DMD-23-01 — Extract a reusable Durability Schedule panel

Completed: 2026-06-10

## Summary

Extracted the Durability Schedule tab UI and state from `client/src/app/database/edit/page.tsx` into `DurabilitySchedulePanel`, a scope-driven component mirroring `AssignChannelsPanel` and `EditMetadataPanel`. The full-page route renders the panel inside the existing tab with `showUploadAffordance={false}` so schedule upload stays in the side panel; the panel includes inline upload for dialog embedding in DMD-23-02.

Schedule load, row hydration, inline edit, reset, save, attach/extract, dirty reporting via `onDirtyChange`, and write-permission gating match the prior full-page behavior.
