# DB14-08: Round-trip regression test pack for source-truth model

## Summary

Added a consolidated behavior-focused regression pack that exercises the lean source-of-truth model end-to-end through public service interfaces.

## Test file

`tests/server/services/test_roundtrip_regression.py`

## Coverage

| Behavior | Test |
|----------|------|
| CSV ingest → transfer export/import retains source artifact, preview, measurements, LTTB, lineage | `test_csv_roundtrip_preserves_lineage_preview_measurements_and_lttb` |
| RSP ingest retains original + canonical artifacts; downstream parity with CSV path | `test_rsp_roundtrip_preserves_dual_artifacts_and_matches_csv_downstream` |
| YAML vs UI channel-map paths produce equivalent plot column mappings | `test_channel_map_yaml_and_ui_paths_produce_equivalent_snapshots` |
| Durability schedule one-active-per-version, edit permissions, transfer retention | `test_durability_schedule_enforces_one_active_and_edit_permissions` |
| Transfer import rejects checksum mismatch and missing artifacts | `test_transfer_import_rejects_checksum_mismatch_and_missing_artifacts` |

## Approach

Tests use vertical-slice TDD against public APIs (`IngestionService.ingest`, `TransferPackageService.export_package` / `import_package` / `validate_package`, `DurabilityScheduleStorageService.attach_schedule`) rather than private helpers.
