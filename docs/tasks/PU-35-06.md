# PU-35-06 — Cumulative-by-channel 2D renderer

## Behavior added

- Added `CumulativeByChannelPlotCard` at `client/src/features/inspect-damage-3d/components/CumulativeByChannelPlotCard.tsx`.
- The component renders grouped Reference/Target bars from `Damage2DPlotSpec` inside shared `PlotCardShell` chrome.
- Renderer includes Dashboard-style SVG grid, y-axis ticks, channel categories, and a legend for Reference/Target colors.
- Bar hover content is provided through native SVG `<title>` tooltips with channel, dataset, value, value mode, and scale context text.
- Bars expose practical accessibility labels (`aria-label`) for per-bar value context.

## Interfaces changed

- New exported component:
  - `CumulativeByChannelPlotCard`
- New component props:
  - `spec: Damage2DPlotSpec`
  - optional `isLoading`, `error`, and `className`

## Tests added

- `client/src/features/inspect-damage-3d/components/CumulativeByChannelPlotCard.test.tsx`
  - validates title/subtitle + grouped bars/categories rendering,
  - validates legend labels for Reference/Target,
  - validates tooltip text and bar accessibility labels,
  - validates card-shell empty/loading/error state behavior.
