# P14-12 — Replace prompt-based DB switch UX

Date: 2026-06-09  
Status: DONE

## Scope

Replace browser prompts used by database create/connect actions with a typed-confirm dialog and searchable managed-database picker.

## Implementation notes

1. Added `DatabaseSwitchDialog` component under upload UI:
   - shared dialog for both `create` and `connect` modes,
   - searchable DB list for connect flow,
   - explicit expected-confirmation label,
   - submit enablement only when typed confirmation matches expected text.
2. Updated Database page runtime switch handlers:
   - create now opens modal instead of `window.prompt`,
   - connect loads catalog into modal instead of prompt list text,
   - submit continues to call existing backend APIs.
3. Preserved backend safety contract:
   - API still validates typed confirmation server-side.
