# PU-35-11 — Add delta metric toggle

## Behavior added

- Added a subtle card-local segmented toggle in `TargetDeltaVsReferencePlotCard` for `Absolute`, `Percent`, and `Ratio` metric modes.
- Metric selection is local to the delta card UI and does not add or modify global sidepanel controls.
- Percent mode now displays `100 * (target - reference) / reference` and ratio mode displays `target / reference`.
- Low-reference channels are treated as unavailable for percent/ratio modes and render with explicit unavailable copy instead of misleading divide-by-near-zero values.
- Delta card tooltip copy, legend label text, aria labels, and axis/tick formatting now adapt to the active metric mode.

## Interfaces changed

- Added a new pure utility module:
  - `client/src/features/inspect-damage-3d/lib/delta-metric-mode.ts`
  - Exposes `DeltaMetricMode`, `computeDeltaMetricValue`, `computeDeltaMetricDomain`, and `getDeltaMetricLabel` for consistent mode-specific behavior.
- No backend API contracts changed.
- No `DamageComparisonState` persistence shape changed.

## Tests added

- `client/src/features/inspect-damage-3d/__tests__/delta-metric-mode.test.ts`
  - validates absolute/percent/ratio formulas and low-reference suppression behavior,
  - validates domain behavior for diverging and ratio axes.
- `client/src/features/inspect-damage-3d/components/TargetDeltaVsReferencePlotCard.test.tsx`
  - validates card-local metric toggling behavior,
  - validates percent and ratio tooltip/aria/axis behavior and low-reference unavailable rendering.

## Follow-on assumptions for next slice

- `PU-35-12` can assume delta metric toggle behavior is implemented and locally scoped to the target-delta card.
- Future polish/a11y work can use the existing metric utility module as the canonical source for label/value/domain semantics.

## Decisions intentionally unchanged

- No changes to aggregate source formulas in `build-damage-comparison-aggregates.ts`.
- No changes to sidepanel ownership of channels/value mode.
- No changes to focused 3D plot routing or overlay ownership boundaries.
