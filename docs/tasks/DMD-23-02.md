# DMD-23-02 — Add Durability Schedule to the metadata popup dialog

Completed: 2026-06-10

## Summary

Extended `MetadataEditDialog` with a third left-nav item, **Durability Schedule**, directly below **Assign Channels**. The dialog still opens from the version-row pencil, defaults to Edit Metadata, and scopes all three sections to the clicked program/version. `DurabilitySchedulePanel` is mounted with `showUploadAffordance` so schedule upload lives in the dialog content area (unlike the full-page route, which keeps upload in the side panel). All panels remain mounted while the dialog is open (`hidden` toggles visibility) so in-progress drafts persist across section switches.

Schedule dirty state is wired via `onDirtyChange` but not yet included in the shared discard-close prompt — that is DMD-23-03.
