## Summary

Extracted shared per-event channel resolution for derived-data readers:

- `server/services/per_event_channel_resolver.py` — pure resolver from plot mapping + header/units lists to channel names or structured errors (`missing_headers`, `column_out_of_range`).
- `server/services/event_header_provider.py` — loads headers/units per `event_id`, preferring `event_previews` then falling back to retained `ingestion_artifacts.preview_json` `#TITLES` / `#UNITS`.
