# P17-02 — Admin delete managed DB + remove connect typed confirmation

Date: 2026-06-09  
Status: DONE

## Scope

- Remove `CONNECT <filename>` typed confirmation from connect/switch flow.
- Add admin-only delete for non-active managed `dashboard*.db` files with `DELETE <filename>` confirmation.

## Implementation notes

1. Backend (`server/routers/export.py`):
   - `ConnectDatabaseRequest` now only carries `database_name`.
   - New `POST /api/v1/export/database/delete` (admin-only) with typed confirmation and active-DB guard.
2. Frontend:
   - Connect dialog: select + **Connect & Switch** without typing.
   - Admins see trash icon on non-current databases; delete mode shows typed confirmation.
3. Tests updated in `tests/server/routers/test_export_router.py`.

## Verification

- Connect works without confirmation string.
- Delete rejects wrong confirmation, active DB, and unauthenticated callers.
- Delete removes file and refreshes catalog in UI.
