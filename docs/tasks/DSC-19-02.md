## Summary

Replaced first-segment / glob-first-match logic in `build-durability-schedule-rows.ts` with a pure client module mirroring `notebooks/rsp_file_name_extraction_v2.ipynb`.

## Public interface

| Function | Behavior |
|----------|----------|
| `discoverEventDelimiter(fileNames)` | Counts underscore tokens across stems (skip empty + per-file duplicates); picks max `(file_count, -first_position)`. |
| `rspEventNameFromFile(sourceFile, delimiterToken)` | Joins tokens before delimiter; falls back to full stem when delimiter absent or null. |
| `matchSchedulePattern(stem, patterns)` | Longest bare pattern where `pattern in stem`. |
| `buildDurabilityScheduleRows(events, entries)` | Discovers delimiter once, matches each event, sorts by schedule sequence then filename, wraps display patterns with `*` when absent. |
