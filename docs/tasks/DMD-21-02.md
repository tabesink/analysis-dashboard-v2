# DMD-21-02 — Add Assign Channels to the Database metadata dialog

Completed: 2026-06-10

## Summary

Extended `MetadataEditDialog` with a second left-nav item, **Assign Channels**, alongside **Edit Metadata**. The dialog still opens from the version-row pencil, defaults to Edit Metadata, and scopes both sections to the clicked program/version. Both panels remain mounted while the dialog is open (`hidden` toggles visibility) so in-memory drafts persist across section switches. Combined metadata + channel-map dirty state feeds the existing discard-close and pending scope-change prompts.

Section nav styling and active-state logic live in `metadata-dialog-sections.ts`.
