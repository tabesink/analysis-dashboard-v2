# DSC-19-04 — PUT save schedule edits API

**Status:** DONE (2026-06-10)  
**Issue:** `docs/brainstorm/19_durability_sch_impl/issues/DSC-19-04-save-api.md`

## Summary

Added `PUT /api/v1/dashboard/program-version/schedule` for persisting edited table state on the active schedule artifact's `parse_preview_json`. Original `.sch` bytes remain immutable.

## API

- **Request:** `DurabilityScheduleSaveRequest` — `program_id`, `version`, `multiplier`, `event_rows[]`, optional `delimiter_token`
- **Response:** `DurabilityScheduleContextResponse` (same as GET)
- **Auth:** `WriteUserDep` + `user_can_edit_program_version`
- **Errors:** 404 no active schedule; 403 non-owner; 400/422 invalid payload or unknown `event_id`
