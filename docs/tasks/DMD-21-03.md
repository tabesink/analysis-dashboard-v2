# DMD-21-03 — Harden cross-section dirty state, permissions, refresh, and accessibility

Completed: 2026-06-10

## Summary

Made the two-section metadata dialog production-ready for Assign Channels. Combined metadata + channel-map dirty state drives close and pending scope-change prompts with generic "unsaved changes" copy. Channel-map saves flow through `saveProgramVersionChannelMap` → existing `dashboardApi.saveChannelMap` contract → `invalidateQueriesAfterChannelMapSave`, which refreshes channel-map editor data plus all database-table and dashboard filter queries. Write permission continues to gate both panels consistently with the full-page route.
