# P15-03 Lean Plot-Channel Damage Refactor

## Summary

Inspect Damage now calculates against the 12 canonical channels derived from the existing 8 plot-pair channel-map entries. Processed data uses core tables only: `dim_channel_map` resolves the mapped source headers and `measurements_raw.channel_name` provides the raw time-series values.

## Implementation Notes

- Added `server/services/damage_channels.py` to keep the fixed 12-channel mapping and shared-X validation isolated.
- Refactored damage querying to use semantic keys (`bj_x_force`, `bushing_f_z_momt`, etc.) and canonical labels instead of `ChNN` rows.
- Removed damage-specific raw measurement columns from declared schema and insert/select paths.
- Removed full-channel artifact backfill scripts and service.
- Added startup repair for legacy generic `dim_channel_map.x_channel/y_channel` values using retained artifact preview headers.
