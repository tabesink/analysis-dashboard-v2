# P15-01 Inspect Damage Vertical Slice

## Summary

Implemented the first Inspect Damage workflow:

- Store the first 21 full-resolution RSP data channels as `Ch01`-`Ch21` in DuckDB during ingestion.
- Calculate fatigue damage on demand using the notebook-defined `py_fatigue` rainflow + DNV Curve C Palmgren-Miner method.
- Add `/api/v1/damage/inspect` for authenticated compute-on-read requests.
- Add `/inspect-damage` side-panel selection and flat damage table UI.
- Add a one-time backfill helper for existing retained uploads.
