# DMD-20-02 — Database table pencil + metadata edit dialog

Completed: 2026-06-10

## Summary

Replaced version-row event-count text in `DatabaseEventTree` with a compact pencil action. Clicking opens `MetadataEditDialog`, a settings-style modal (overlay, left nav with **Edit Metadata**, right panel) that renders the reusable `EditMetadataPanel` pre-scoped to the clicked program/version. Dialog state lives in `metadata-edit-dialog-store`; opening/closing does not navigate or mutate table filters, expansion, or selection.
