## This slice changes

Add the first target delta versus reference card using signed absolute delta by channel. Positive values mean Target exceeds Reference; negative values mean Target is lower than Reference.

This issue should carry low-reference warnings in the spec, but percent/ratio metrics are implemented in `PU-35-11`.
